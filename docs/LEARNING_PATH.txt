================================================================================
AGENTIC RAG SYSTEM - COMPREHENSIVE LEARNING PATH
================================================================================

This document outlines the recommended learning path to understand all concepts
needed to develop the Agentic RAG system project.

================================================================================
SECTION 1: FOUNDATION - TYPESCRIPT & TESTING (Week 1-2)
================================================================================

Start here if you need to strengthen your TypeScript and testing fundamentals.

1.1 TypeScript Fundamentals
    - Official TypeScript Handbook (Free)
    - https://www.typescriptlang.org/docs/
    - Duration: Self-paced
    - Topics: Types, interfaces, generics, advanced types

1.2 Vitest Fundamentals
    - Vitest Fundamentals Guide (Free)
    - https://agentfactory.panaversity.org/docs/TypeScript-Language-Realtime-Interaction/testing-typescript-ai/vitest-fundamentals
    - Duration: 2-3 hours
    - Topics: Test runner setup, native TypeScript support, Jest-compatible API

1.3 Property-Based Testing with fast-check
    - Property-Based Testing with TypeScript & fast-check (Free Article)
    - https://zenn.dev/samuraikun/articles/property-based-testing-in-typescript
    - Duration: 1-2 hours
    - Topics: Generating test inputs, property assertions, edge cases

1.4 Test-Driven Development with Vitest
    - Test-Driven Development (TDD) with TypeScript and ReactJS using Vitest (Free)
    - https://medium.com/@rmbagt/test-driven-development-tdd-with-typescript-and-reactjs-using-vitest-7187d9126a0e
    - Duration: 2-3 hours
    - Topics: TDD workflow, async testing patterns, mocking

================================================================================
KEY TOPICS TO FOCUS ON
================================================================================

For your specific Agentic RAG project, prioritize these topics:

✓ RAG Pipeline Architecture
  - Document parsing (PDF, DOCX, TXT, MD)
  - Text splitting and chunking
  - Embedding generation
  - Vector storage and retrieval
  - Metadata management

✓ Vector Embeddings & Similarity Search
  - How embeddings work
  - Dimension validation
  - Similarity metrics (cosine, euclidean)
  - Filtering and metadata queries
  - Collection management

✓ LangChain Agents & Tools
  - ReAct agent pattern
  - Tool definitions and execution
  - Agent memory and session management
  - Error handling and sanitization
  - Citation extraction

✓ Next.js API Routes & Streaming
  - API route handlers
  - Server-Sent Events (SSE)
  - Streaming responses
  - Error handling and validation
  - CORS and security

✓ TypeScript & Testing
  - Type safety for RAG components
  - Property-based testing with fast-check
  - Unit tests for parsers and splitters
  - Integration tests for pipeline
  - Mocking external services

✓ Document Management
  - File upload handling
  - Metadata storage
  - Document deletion and cleanup
  - Collection isolation

NOTES
================================================================================

1. Udemy courses often have discounts (50-90% off). Wait for sales or use
   coupon codes provided in course links.

2. Free resources are marked with (FREE) - prioritize these if budget is limited.

3. The learning order is flexible - you can parallelize some sections.

4. Revisit courses during implementation when you encounter specific problems.

5. Practice by building small projects after each section:
   - After RAG section: Build a simple document Q&A system
   - After Vector DB section: Build a semantic search engine
   - After Next.js section: Build a simple API with streaming
   - Final: Combine all into your agentic RAG system

6. Don't try to learn everything before coding. Start coding early and
   reference courses as needed.

7. The project in docs/tasks.md is your ultimate learning goal - use it
   as a guide for what to focus on.
